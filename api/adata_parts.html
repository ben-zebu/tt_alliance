std::string EcE_type = "mean"

Type of temperature dependence for \( \frac{E_c}{E} \) calculation. The expected values are:

  • “none” : no temperature dependence

  • ”mean” : mean value of 2 timesteps

  • ”min” : minimum value of 2 timesteps

  • ”max” : maximum value of 2 timesteps

std::unordered_map<std::string, std::pair<std::string, double>> material_parameters_input

Map used to store the input values for all coefficients that defined the material. The different keys are:

  • “EE” : Young modulus \( E \)

  • “SM” : Admissible stress \( S_m \)

  • “SU” : Ultimate tensile stress \( S_u \)

  • “SCY” : Cyclic yield stress \( S_{cy} \)

  • “SY” : Conventionnal yield stress \( S_y \)

  • “SYG” : Conventionnal yield stress for a large number of cycles \( S_{yg} \)
