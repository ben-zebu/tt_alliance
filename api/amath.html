class Combination
#include <Combination.h>

Class used to manage all possible ranks’ combinations given by a vector of ranks.

This class is used to manage the ranks’ combination identification given by a vector of ranks. The naive represention of all combinations is a square matrix storage with a size equal to vector of ranks. Each element of the matrix represents a combination of ranks. The associated ranks to a combination is identified by the row and column indices. Several cases are possible:

  • The total square matrix if the expected results between (i, j) ranks is different than (j, i) ranks.

  • the triangular matrix if the expected results between (i, j) ranks is equal to (j, i) ranks.

  • The superior triangular matrix if the expected results between (i, j) ranks is equal to (j, i) ranks and the the combination (i, i) is excluded.

Subclassed by amath::SquareCombination, amath::TriangularCombination

class TriangularCombination : public amath::Combination
#include <Combination.h>

Class used to manage all possible ranks’ combinations given by a vector of ranks.

All combination are given by a triangular matrix representation with a size equal to vector of ranks. The allowed cases are:

  • without diagonnal terms and the following matrix:

    \[\begin{split}\begin{eqnarray*} \left [ \begin{array}{cccc} na & (0,1) & ... & (0,n-1) \\ na & na & ... & (1,n-1) \\ na & na & na & (2,n-1) \\ na & na & na & na \end{array} \right ] \end{eqnarray*}\end{split}\]

  • with diagonnal terms and the following matrix:

    \[\begin{split}\begin{eqnarray*} \left [ \begin{array}{cccc} (0,0) & (0,1) & ... & (0,n-1) \\ na & (1,1) & ... & (1,n-1) \\ na & na & (2,2) & (2,n-1) \\ na & na & na & (n-1,n-1) \end{array} \right ] \end{eqnarray*}\end{split}\]

constexpr std::size_t combination_for_line(const std::size_t &p, const std::size_t n) const

Determine the number of combinations stored in the first \( p \) lines.

Determine the number of combinations stored in the first \( p \) lines based on the following formula:

\[ S(p,n) = \sum_{i=0}^{p} d(n) - i = \sum_{j=d(n)-p}^{d(n)} j = \frac{1}{2} (p+1)(2d(n)-p) \]
with :

  • \( d(n) = n \) if diagonal is set,

  • \( d(n) = n - 1\) if diagonal is not set. The \( d(n) \) result is given by function diag_shift()

Parameters:

  • p – Number of lines.

  • n – Number of ranks.

Returns:

First combination represented on the p line.

constexpr std::size_t line_for_combination(const std::size_t &c, const std::size_t n) const

Determine the line associated to a combination.

Return the \( p \) line associated to the combination \( c \) such as: \( S(p-1,n) \leq c \leq S(p,n) \) where \( S(p,n) \) gives the number of combination stored in the \( p \) lines (see combination_for_line()). The \( p \) line is the upper integer associated to the lower root of the following polynom:

\[ p^2 + b.p + 2(c+1) \]
with \( b = 1 +2 d(n) \) and \( d(n) \) the result is given by function diag_shift()

Parameters:

  • c – combination number.

  • n – Number of ranks.

Returns:

line number associated to the combination c.

double tresca() const

Calculates the Tresca stress, also known as the maximum shear stress.

The Tresca stress is the maximum absolute difference between the principal stresses. It is determined with the following formula: \(\sigma_\text{Tresca} = \max\left(\sigma_1 - \sigma_2, \sigma_2 - \sigma_3, \sigma_3 - \sigma_1\right)\). where \(\sigma_1\), \(\sigma_2\), and \(\sigma_3\) are the principal stresses.

double mises() const

Calculates the von Mises stress.

The von Mises stress is calculated from the principal stresses using the following formula: \(\sigma_\text{von Mises} = \sqrt{\frac{1}{2}\left[(\sigma_1 - \sigma_2)^2 + (\sigma_2 - \sigma_3)^2 + (\sigma_3 - \sigma_1)^2\right]}\) where \(\sigma_1\), \(\sigma_2\), and \(\sigma_3\) are the principal stresses.

double reduced_mises() const

Calculates the reduced von Mises stress. It is a homothetic transformation of the von Mises / stress with the factor \(\frac{2}{\sqrt{3}}\).

Constant factor used in the calculation of an equivalent stress based on an homothecy from von Mises stress. The von Mises stress is calculated as \(\sigma_\text{VM} = \sqrt{\frac{2}{3}\left(\sigma_1^2 + \sigma_2^2 + \sigma_3^2\right)}\), where \(\sigma_1\), \(\sigma_2\), and \(\sigma_3\) are the principal stresses. class TorsorCombination
#include <TorsorCombination.h>

Class used to manage the torsor’s combinations. All possible torsor combination are represented by a binary tree. The following representation is given for 4 torsors, the second one is constant. 

///                                   tree_rk
///                           / Max      0
///                     / Max   
///                    /     \ min       1
///       / Max --- cst 
///      /             \     / Max       2
///     /               \ min
///    /                     \ min       3
///   *
///    \                     / Max       4
///     \               / Max
///      \             /     \ min       5
///       \ min --- cst
///                    \     / Max       6
///                     \ min 
///                          \ min       7
///
The number of torsor combination is equal to \( 2^t \) where \( t \) is the number of torsor with different. coefficients.

For a given torsor combination, the choice between the maximum and the minimun value is based on the parity of each explorer branch. For the 5th combination of the example, the retained coefficients are:

  • 4th torsor associated to \( tree\_rk = 5 \rightarrow \) minimal coefficient

  • 3rd torsor associated to \( tree\_rk = 5//2 = 2 \rightarrow \) maximal coefficient

  • 2nd torsor associated to \( tree\_rk = 2 \rightarrow \) constant coefficient

  • 1st torsor associated to \( tree\_rk = 2//2 = 1 \rightarrow \) minimal coefficient
